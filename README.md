# ToDoTogehter

ToDoTogether 프로젝트의 Front-End Repository입니다.

[Back-End Repository 바로가기](https://github.com/nuyh99/WAP_Todo)

## 구성원들과 함께 ToDoList를 작성하자!

### 구성

- 방을 생성해서 팀원을 초대할 수 있습니다.
- 초대코드를 통해 다른 방에 참가할 수 있습니다.

### 사용한 라이브러리(패키지)

- React
  - react-router-dom
  - mui
  - svg-icons
  - react-beautiful-dnd
  - uuid
  - redux, react-redux
  - axios
  - date-fns
  - react-loader-spinner
